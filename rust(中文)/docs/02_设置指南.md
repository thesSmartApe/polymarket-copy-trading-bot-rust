## 1. 先决条件

在开始之前，请确保您有：

- 运行 Windows、macOS 或 Linux 的计算机
- 互联网连接
- 文本编辑器（记事本、VS Code 或任何文本编辑器）
- 基本的计算机技能（打开文件、复制文本）

## 2. 安装 Rust

### 2.1 Windows

1. 访问 https://rustup.rs/
2. 下载安装程序（`rustup-init.exe`）
3. 运行安装程序
4. 出现提示时，按 `Enter` 继续默认安装
5. 安装后重启终端/PowerShell

**验证安装：**
打开 PowerShell 并输入：
```powershell
rustc --version
```
您应该看到类似 `rustc 1.xx.x` 的内容。

### 2.2 macOS

打开终端并运行：
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

按照提示操作（按 `Enter` 使用默认值）。

**验证安装：**
```bash
rustc --version
```

### 2.3 Linux

打开终端并运行：
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

按照提示操作。

**验证安装：**
```bash
rustc --version
```

## 3. 设置您的钱包

### 3.1 选项 1：创建新钱包（推荐用于测试）

1. 安装 [MetaMask](https://metamask.io/) 浏览器扩展
2. 创建新钱包（或使用现有钱包）
3. **添加 Polygon 网络：**
   - 点击网络下拉菜单（左上角）
   - 点击"添加网络"
   - 填写：
     - 网络名称：`Polygon Mainnet`
     - RPC URL：`https://polygon-rpc.com`
     - 链 ID：`137`
     - 货币符号：`MATIC`
     - 区块浏览器：`https://polygonscan.com`

4. **获取您的私钥：**
   - 点击账户图标（右上角）
   - 点击"账户详情"
   - 点击"导出私钥"
   - 输入您的密码
   - **复制私钥**（这是您的 `PRIVATE_KEY` - 请保密！）
   - 如果存在，请删除 `0x` 前缀

5. **获取您的地址：**
   - 您的地址显示在账户名称下方
   - 看起来像：`0x1234...5678`
   - 复制此地址（这是您的 `FUNDER_ADDRESS`）

6. **为钱包充值：**
   - 您需要在 Polygon 上有 USDC 或 USDC.e
   - 从以太坊桥接代币到 Polygon，或在交易所购买
   - 建议最低金额：$50-100 用于测试

### 3.2 选项 2：使用现有钱包

如果您已经在 Polygon 上有资金的钱包：
1. 导出您的私钥（见上文）
2. 获取您的钱包地址
3. 确保您有 USDC/USDC.e 用于交易

## 4. 获取 API 密钥

机器人需要与 Polygon 区块链建立 WebSocket 连接。您将使用 Alchemy 或 Chainstack。

### 4.1 选项 1：Alchemy（推荐）

1. 访问 https://www.alchemy.com/
2. 点击"创建应用"或"注册"
3. 填写：
   - 应用名称：`Polymarket Bot`
   - 链：`Polygon`
   - 网络：`Polygon Mainnet`
4. 创建后，点击您的应用
5. 找到"API 密钥"部分
6. 复制 API 密钥（这是您的 `ALCHEMY_API_KEY`）

**免费套餐包括：** 每月 300M 计算单位（对此机器人来说绰绰有余）

### 4.2 选项 2：Chainstack（替代方案）

1. 访问 https://chainstack.com/
2. 注册免费账户
3. 创建新项目
4. 添加 Polygon Mainnet 节点
5. 获取您的 WebSocket URL
6. 从 URL 中提取 API 密钥（这是您的 `CHAINSTACK_API_KEY`）

## 5. 查找鲸鱼地址

"鲸鱼"是您想要复制的成功交易者。以下是查找方法：

### 5.1 方法 1：Polymarket 排行榜

1. 访问 https://polymarket.com/leaderboard
2. 寻找胜率高和利润高的交易者
3. 点击交易者的个人资料
4. 找到他们的钱包地址（通常在他们的个人资料中可见）
5. 复制地址（40 个字符，如果存在则删除 `0x`）

### 5.2 方法 2：分析最近的赢家

1. 前往 Polymarket 市场
2. 查看"已结算"市场
3. 找到有大额支付的市场
4. 点击获胜持仓
5. 记录持有获胜持仓的钱包地址
6. 研究这些地址以找到持续获胜者

### 5.3 方法 3：社交媒体/社区

- 查看 Polymarket Discord/Telegram
- 寻找分享其地址的交易者
- 在复制之前验证他们的交易记录

**重要提示：** 在复制之前，请始终验证鲸鱼的表现。过去的表现不能保证未来的结果。

## 6. 配置机器人

### 6.1 步骤 1：克隆/下载仓库

如果您有 git：
```bash
git clone https://github.com/soulcrancerdev/polymarket-kalshi-copy-trading-arbitrage-bot
cd polymarket-kalshi-copy-trading-arbitrage-bot/rust
```

或下载并解压 ZIP 文件。

### 6.2 步骤 2：创建您的 .env 文件

**Windows (PowerShell)：**
```powershell
Copy-Item .env.example .env
```

**macOS/Linux：**
```bash
cp .env.example .env
```

**或手动：**
1. 复制 `.env.example`
2. 将副本重命名为 `.env`（Linux/macOS 上无扩展名）

### 6.3 步骤 3：编辑 .env 文件

在任何文本编辑器中打开 `.env`。您会看到类似以下内容：

```env
PRIVATE_KEY=your_private_key_here
FUNDER_ADDRESS=your_wallet_address_here
TARGET_WHALE_ADDRESS=target_whale_address_here
ALCHEMY_API_KEY=your_alchemy_api_key_here
```

替换每个值：

1. **PRIVATE_KEY**：粘贴您的钱包私钥（无 `0x` 前缀）
2. **FUNDER_ADDRESS**：粘贴您的钱包地址（可以有 `0x` 或没有）
3. **TARGET_WHALE_ADDRESS**：粘贴鲸鱼地址（无 `0x` 前缀）
4. **ALCHEMY_API_KEY**：粘贴您的 Alchemy API 密钥

**示例（不要使用这些 - 它们是假的）：**
```env
PRIVATE_KEY=0123456789abcdef0123456789abcdef0123456789abcdef0123456789abcdef
FUNDER_ADDRESS=0x1234567890123456789012345678901234567890
TARGET_WHALE_ADDRESS=204f72f35326db932158cba6adff0b9a1da95e14
ALCHEMY_API_KEY=abc123xyz789
```

### 6.4 步骤 4：设置初始交易模式

对于首次运行，请设置：
```env
ENABLE_TRADING=false
MOCK_TRADING=true
```

这可以让您看到机器人会做什么，而无需实际交易。

## 7. 测试您的设置

### 7.1 步骤 1：验证配置

运行配置检查器：
```bash
cargo run --release --bin validate_setup
```

**它检查的内容：**
- 所有必需值都已设置
- 地址格式正确
- API 密钥格式有效
- 私钥格式正确

**在继续之前修复它报告的任何错误。**

### 7.2 步骤 2：构建机器人

```bash
cargo build --release
```

第一次构建需要 5-10 分钟（下载依赖项）。

### 7.3 步骤 3：测试运行（模拟模式）

确保您的 `.env` 有：
```env
ENABLE_TRADING=false
MOCK_TRADING=true
```

然后运行：
```bash
cargo run --release
```

**预期结果：**
- 机器人连接到区块链
- 您看到连接消息
- 当鲸鱼交易时，您看到模拟交易消息
- 不会下实际订单

**如果您看到错误：**
- 查看 [故障排除指南](06_故障排除.md)
- 验证您的 API 密钥是否正确
- 确保您的地址格式正确

## 8. 运行机器人

### 8.1 步骤 1：启用交易

一旦您测试过并确信：

1. 编辑 `.env`
2. 设置：
   ```env
   ENABLE_TRADING=true
   MOCK_TRADING=false
   ```

3. 保存文件

### 8.2 步骤 2：运行机器人

```bash
cargo run --release
```

**Windows 用户：** 您也可以使用 `run.bat`（设置后双击）。

### 8.3 步骤 3：监控输出

您会看到类似以下的消息：
```
🚀 Starting trader. Trading: true, Mock: false
🔌 Connected. Subscribing...
⚡ [B:12345] BUY_FILL | $100 | 200 OK | ...
```

**每条消息的含义：**
- `[B:12345]` = 检测到交易的区块号
- `BUY_FILL` = 交易类型（BUY 或 SELL）
- `$100` = 鲸鱼交易的美元价值
- `200 OK` = 您的订单已成功下单
- 后面的数字 = 您的成交详情

### 8.4 步骤 4：检查结果

- **实时：** 实时观看控制台输出
- **CSV 日志：** 检查 `matches_optimized.csv` 以查看所有交易
- **Polymarket：** 在 Polymarket 网站上检查您的持仓
